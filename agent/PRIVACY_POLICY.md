# Privacy Policy for DarkCode Agent

**Last updated: December 19, 2024**

## Overview

DarkCode Agent is a remote control application for Claude Code CLI. This privacy policy explains how the app handles your data.

## Data Collection

**DarkCode Agent does NOT collect, store, or transmit any personal data to external servers.**

### What the app accesses:
- **Network access**: To connect to your self-hosted DarkCode Server via WebSocket
- **Camera access**: Only for QR code scanning to configure server connections (optional)

### What is stored locally on your device:
- Server connection details (IP address, port, authentication token)
- Chat history (stored locally in app database)
- User preferences (theme, settings)

### What is NOT collected:
- No analytics or tracking
- No advertising identifiers
- No personal information
- No location data
- No contacts or phone data

## Data Transmission

All data transmitted between the app and your DarkCode Server:
- Stays on YOUR network (LAN, VPN, or SSH tunnel)
- Is encrypted via TLS/WSS when enabled
- Never passes through third-party servers
- Is controlled entirely by you

## Third-Party Services

DarkCode Agent does not use any third-party services, analytics, or advertising SDKs.

## Data Storage

All data is stored locally on your device. Uninstalling the app removes all stored data.

## Children's Privacy

This app is intended for developers (18+) and is not designed for children under 13.

## Changes to This Policy

We may update this privacy policy. Changes will be posted in the app repository.
